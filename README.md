# StarWave  

StarWave is a dynamic web application designed to connect event organizers with influencers. The platform allows users to browse and book influencers for various events while enabling influencers to create detailed profiles and manage client bookings efficiently.  

## Features  

- **User & Influencer Profiles**: Users can search for and book influencers, while influencers can register, create profiles, and manage bookings.  
- **Intuitive Interface**: A user-friendly design that facilitates seamless interactions between users and influencers.  
- **Robust Booking System**: Effectively schedule and manage events with an easy-to-use booking system.  
- **Profile Customization**: Influencers can customize their profiles to showcase their unique styles and experiences.  
- **Real-Time Updates**: Instant updates on booking status and influencer availability.  
- **Secure Registration**: A secure registration process to protect user information.  
- **Search Functionality**: Quickly find specific influencers based on desired criteria.  

## Technologies Used  

- **Frontend**: HTML, CSS, JavaScript, jQuery, AngularJS, Bootstrap, AJAX  
- **Backend**: Node.js, Express  
- **Database**: MySQL
